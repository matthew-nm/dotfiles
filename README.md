dotfiles
========

## ZSH
Add the following to local file `~/.zshrc`
```
# Define local
#typeset -A SYMDIRS
#SYMDIRS[/Volumes/Data/Drive]='/Users/matthew/Drive'

#export MYCOLOR="#71b5c9"

# Source common
source ~/dotfiles/zshrc

## User definitions ##
```
